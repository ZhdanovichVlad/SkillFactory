# <center> Проект №5. Компьютер говорит нет.
Добрый день. Вашему вниманию представлен учебный проект по машинному обучению.    
Ссылка на [kaggle](https://www.kaggle.com/c/sf-dst-scoring) с описанием хакатона.
Цель проекта - построить скоринговую модель, которая должна помочь определить банку клиентов, которые не вернут кредит (дефолтные клиенты).     
Основной метрикой выбрана Roc - кривая. Также использовались и другие основные метрики бинарной классификации, для определения качества модели.       
Для проведения разведочного анализа данных написано несколько функций. На основе графиков и корреляционного анализа, выделены предположительные значимые признаки.     
С помощью f_classif и mutual_info_classif оценили значимость переменных.     
В результате future engineering добавлены новые признаки, которые улучшили показатели модели.  
С помощью подбора гиперпараметров удалось добиться Roc-auc = 0.738.    
Однако выборка не сбалансированная. Balanced accuracy равна всего 0.506.    
Судя по Confusion matrix построенная модель не совсем хорошо распознаёт дефолтных клиентов.    
Необходимо провести дополнительные исследования.
